# ①課題名
補助金マッチング判定システム(サブ)

## ②課題内容（どんな作品か）
PHPを使用した補助金適合度判定システム
見た目などを改良

### 主な機能
1. 補助金マッチング判定
   - 企業の情報や事業計画を入力すると、各補助金との適合度を0〜100点のスコアで判定
   - 4段階評価（非常に適合/適合/やや適合/要検討）で表示
   - ものづくり補助金と事業再構築補助金の2種類に対応（プロトタイプ）

2. セッション&ログイン機能
   - ログイン認証
   - セッション管理によるページ遷移ごとのセッションID再生成

3. ユーザー管理機能
   - ユーザーの登録・一覧表示・更新・削除

4. 判定履歴管理
   - 過去の判定結果を一覧表示
   - 判定詳細の確認
